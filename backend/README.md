# Backend Flask API for domain claim and Namecheap integration
# 1. Set your Namecheap API credentials in backend/.env
# 2. Install dependencies: pip install -r requirements.txt
# 3. Run: python app.py
#
# The /api/claim-domain endpoint expects JSON: {"domain": "example.com", "email": "user@email.com"}
#
# For production, deploy with gunicorn or similar and secure your .env file.
